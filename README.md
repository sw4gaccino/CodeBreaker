# CodeBreaker

Ein Passwort-Ratespiel in Java. Man bekommt Hinweise und muss das versteckte Passwort erraten. Wird mit jedem Level länger und schwieriger.

## Wie es funktioniert

Du bekommst am Anfang ein paar Hinweise – z.B. wie lang das Passwort ist, ob Zahlen oder Buchstaben drin sind usw. Mit jedem Level kommen mehr Zeichen und weniger Versuche dazu. Ab Level 8 kommen auch Großbuchstaben, ab Level 12 Sonderzeichen.

Wenn du nicht weiterkommst kannst du `hinweis` eintippen, kostet aber einen Versuch.

## Starten

```bash
javac CodeBreaker.java
java CodeBreaker
```

Die Dateien in einem Ordner haben und in Powershell die oben gennanten Befehle ausführen. (JDK muss installiert sein)

## Technologien

Java, Konsolenanwendung, OOP
